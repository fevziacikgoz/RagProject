# Teknolojiler ve Projeler

Backend ekibi C# .NET, Java Spring Boot ve Go dillerini kullanır. Mesajlaşmada RabbitMQ ve Apache Kafka, veritabanı olarak PostgreSQL ve MongoDB tercih edilir.
Frontend ekibi React, Next.js ve TypeScript kullanır. Mobil ekip React Native ile çapraz platform, native tarafta ise Swift ve Kotlin ile geliştirme yapar.
Veri ve Yapay Zeka ekibi Python, PyTorch, scikit-learn ve Apache Spark kullanır. LLM tabanlı projelerde RAG mimarisi, pgvector ve OpenAI API'leri tercih edilir.
DevOps ekibi Kubernetes, Docker, Terraform ve GitHub Actions ile CI/CD süreçlerini yönetir. Bulut altyapısı Amazon Web Services (AWS) üzerinde çalışır.
Aktif projeler: Atlas adlı kurumsal veri platformu, Pulse adlı gerçek zamanlı analitik ürünü ve Sage adlı yapay zeka destekli müşteri asistanı. Atlas ekibinde 24, Pulse ekibinde 16, Sage ekibinde 12 kişi çalışır.
Her çalışana MacBook Pro veya yüksek konfigürasyonlu Windows dizüstü bilgisayar, ek monitör ve aksesuarlar sağlanır. Donanım yenileme döngüsü 3 yıldır.
